# AGENTS.md

## Project
Build a premium single-page portfolio website for Shagun Tiwari, Product Manager.

## Primary objective
Transform the existing Notion portfolio into a highly polished, mobile-compatible, animated, premium website inspired structurally by kanesherwell.com but visually more immersive and custom.

## Source files
- Read `task-graph.md` for phased execution plan.
- Read `content/shagun-source.md` for portfolio content.
- Read `content/brand-direction.md` for visual direction.

## Non-negotiables
- Single-page site only.
- Use modals/drawers for deep case study views if needed.
- Mobile-first and production-grade.
- No generic AI-template aesthetics.
- Strong motion quality, but preserve performance.
- Accessibility and reduced-motion support required.
- No fake metrics or invented experience details.

## Preferred stack
- Next.js latest
- React latest
- TypeScript
- Tailwind CSS
- Framer Motion
- GSAP only if necessary
- React Three Fiber only for meaningful hero interaction

## Build rules
- Plan before coding.
- Create and validate design system before building sections.
- Implement in phases from `task-graph.md`.
- Keep code modular and reusable.
- Test desktop and mobile.
- Keep animations tasteful and optimized.

## Deliverables
- Working single-page portfolio
- Reusable component structure
- Data/content-driven case study sections
- Clear README
